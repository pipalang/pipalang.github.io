Programming as fun as flying kite

fn main(): i32 {
  return fib(13);
}

fn fib(n: i32): i32 {
  if n <= 1 {
    return n;
  }
  return fib(n - 1) + fib(n - 2);
}

Testing it

First, you need to clone and build the Pipa compiler (pipac):

git clone https://git.sr.ht/~johnnyrichard/pipac
make all

Next, look at the examples folder and build them with pipa cli

./pipa examples/main.pipa
